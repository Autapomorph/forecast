# Weather forecast sample app

## How to run?
- configure `.env` file (see `.env.example`)
- run `yarn`
- run `yarn start` or separately run `yarn start:server` & `yarn start:client`

## How to build?
- configure .env file (see .env.example)
- run `yarn build`
- run `serve -s build`
- run `yarn start:server`
